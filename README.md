# Projetos MicroServices com Spring

Este repositório contém uma série de projetos desenvolvidos para o aprendizado de micro serviços utilizando o framework Spring. Cada pasta representa uma aula ou um projeto específico com diferentes conceitos e práticas relacionadas ao desenvolvimento de aplicações Spring.

## Conteúdo do Repositório

A seguir, você encontrará uma breve descrição de cada um dos diretórios contidos neste repositório:

- **Aula02-Servlet+JSPWorkSpace**: Exercícios iniciais com Servlet e JSP.
- **Aula03-MVCSpringAndThymeleaf**: Introdução ao Spring MVC com Thymeleaf.
- **Aula04-produtoMVC**: Desenvolvimento de uma aplicação MVC para gestão de produtos.
- **Aula05-produtoMVCContinuação**: Continuação do projeto de produto MVC.
- **Aula06-MVCSpringExAluno**: Exemplo de aplicação MVC demonstrando a gestão de alunos.
- **Aula07-MVCSpringExAlunoH2**: Uso da base de dados H2 na aplicação MVC de alunos.
- **Aula07-MVCSpringExProduto**: Exemplo de aplicação MVC para produtos.
- **Aula08-produto-mvc**: Projeto finalizado de produto utilizando MVC.
- **Aula10-lojaExerc**: Exercício prático de uma loja online.
- **Aula11-boasPraticas**: Abordagem de boas práticas no desenvolvimento com Spring.
- **Aula12-categoria**: Aplicação de gerenciamento de categorias.
- **Aula13-categoria+produto**: Integração entre gerenciamento de categorias e produtos.
- **Aula14-lombok+oracleDataBase**: Utilização do Lombok e integração com Oracle Database.
- **Aula15-servicos**: Implementação de serviços em uma aplicação Spring.
- **Aula18-19/mvc-usuario**: Desenvolvimento e complemento de uma aplicação MVC para a gestão de usuários.
- **Aula20**: Continuação de implementações e aprimoramentos.
- **Aula21/produto-mvc_aula_16**: Revisão e aprimoramento da aplicação de produtos MVC.
- **Aula22/produto-mvc**: Versão atualizada e otimizada do projeto de produto MVC.
- **api-produto_aula_28**: Novo projeto de API REST para gestão de produtos.

## Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/ProjetosMicroServices.git
   ```
   
2. Navegue até o diretório de interesse e siga as instruções específicas de cada projeto.

## Pré-requisitos

- Java JDK 8 ou superior
- Maven
- Spring Framework
- Banco de Dados (conforme especificado em cada projeto)
